# こんだての森
あなたの毎日の献立を提案します

<img width="1440" alt="スクリーンショット 2020-11-30 15 16 47" src="https://user-images.githubusercontent.com/63761544/102864612-ca413480-4477-11eb-906c-01a103c74b70.png">

## 機能一覧
- Googleログイン
- My献立の編集・削除・投稿
- 曜日ごとの献立表示切り替え
- algoliaによるメニュー検索
- ユーザー情報編集
- ツアー形式のチュートリアル
- 無限スクロールによる追加検索
- 楽天APIによるレシピ表示
- 退会処理
